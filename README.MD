🌍 Baza Grunt Osnova — Group Travel Organizer
📌 Overview

Baza Grunt Osnova is a full-stack web application designed to simplify the process of organizing group trips.

Planning trips with multiple people can be chaotic — budgeting, task distribution, deadlines, and coordination often become difficult.
This platform helps groups collaborate efficiently, manage responsibilities, and stay organized throughout the entire trip planning process.

✨ Key Features
    👥 Group Trip Management – create and manage trips with multiple participants
    📋 Task Assignment – distribute responsibilities between group members
    ⏰ Deadlines Tracking – keep track of important dates and tasks
    💰 Budget Coordination – manage shared expenses

🚀 Tech Stack
    Backend
        Laravel (REST API)
    Frontend
        React (TypeScript)
    Infrastructure & Deployment
        Docker
        Render (hosting)
        PostgreSQL (managed database)

🏗️ Architecture
The project follows a client-server architecture:
    React frontend interacts with the backend via API
    Laravel backend handles business logic and data processing
    PostgreSQL database stores application data
    Docker ensures consistent deployment across environments

🌐 Deployment
    The application is deployed on Render using Docker containers.
